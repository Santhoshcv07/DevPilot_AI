# 04.02 Clean Architecture & Routers

> **The Mission:** Prevent monolithic code by separating our API endpoints into modular, dedicated files using FastAPI's `APIRouter`.

## 1. Mental Model & Architecture
* **Separation of Concerns (SoC):** A design principle where a program is divided into distinct sections, each handling a specific task. (e.g., The waiter shouldn't be cooking the food).
* **APIRouter:** A "mini-FastAPI" application. It allows us to group related routes (like all user routes, or all AI routes) in separate files, which are then plugged into the central `main.py` file.

**Current Architecture State:**
```text
DevPilot-AI/
├── backend/
