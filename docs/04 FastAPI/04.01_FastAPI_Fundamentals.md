# 04 FastAPI & Uvicorn Fundamentals

> **The Mission:** Establish the core application server (the "brain") of DevPilot AI to listen for web traffic and process data.

## 1. Mental Model & Architecture
* **The API (FastAPI):** The Waiter. It takes requests from the user, translates them, talks to the kitchen (backend logic/database), and returns the food (data).
* **The Server (Uvicorn):** The Restaurant Building. It provides the infrastructure, electricity, and open doors required for the waiter to do their job 24/7.
* **JSON:** The universal language of the web. FastAPI automatically translates Python dictionaries into JSON text so web browsers can understand the response.
